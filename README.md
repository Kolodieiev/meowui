Meowui — це фреймворк, призначений для розробки прошивок для мікроконтролерів ESP32 та ESP32-S3.

Прошивка збирається плагіном [Pioarduino](https://github.com/pioarduino) та не може бути зібрана з допомогою Platformio. Прошивку налаштовано для плати [Лілка v2.2+](https://lilka.dev/). Щоб зібрати прошивку для власної плати, видаліть/відредагуйте приклади контекстів та налаштуйте параметри в заголовкових файлах в каталозі ```include```.

> [!NOTE]
> Приклади контекстів в цьому репозиторії наразі не актуалізовані і можуть
> містити застарілий код. Їх наявність тут необхідна для демонстрації можливостей
> Meowui на популярній платі ЛілкаV2.
> Оновлені приклади, які будуть підтримуватися в майбутньому, можна знайти в репозиторії [Ідеальної консолі](https://github.com/Kolodieiev/perfect_console).

## Карта пам'яті
Якщо ви хочете перевірити усі можливості прошивки, наприклад, реалізацію mp3 програвача, тоді вам буде необхідно налаштувати правильно деякі каталоги на карті пам'яті.  
mp3 плєер шукає плейлисти в каталозі 
```
music 
|__dir1  
|__dir2
```
Читалка txt файлів, шукає бібліотеки в каталозі
```
books 
|__dir1  
|__dir2
```
Зображення шпалер повинно знаходитися в каталозі wall та мати назву wallpaper.bmp. В майбутньому буде додано вільний вибір зображення.
> [!NOTE]
> Наразі підтримуютья тільки зображення формату bmp-16біт-565.

Тестове зображення для шпалер можна завантажити за цим посиланням
https://github.com/Kolodieiev/lilka_v2_meowui/blob/main/images/wall/wallpaper.bmp  

## Прив'язка кнопок

Для прокрутки списків меню, натискайте або утримуйте кнопки вгору/вниз.  
A - Вибір.  
B - Назад.  
D - Меню/контекстне меню.  
С - Розширена інформація.  
Start - Утримування призведе до підтвердження важливої дії. Наприклад, підтвердження створення каталогу або прейменування файлу.  
Select - Утримування призведе до відміни важливої дії. Наприклад, закриття діалогового вікна зміни імені файлу.  

У грі "Комірник" коротке натискання _С_ дозволяє перемикатися між вільною камерою та персонажем. Довге утримання _Start_ скидає рівень. Утримування _Select_ закриває рівень.
