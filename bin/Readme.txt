1. Встановити pip, якщо не встановлено.

2. Виконати команду у терміналі: 
pip install esptool

3. Підключити Лілку до ПК в режимі прошивки.

4. Виконати команду у терміналі:
python -m esptool --chip esp32s3 flash-id
та переконатися, що МК підключено і його пам'ять читається.

5. Виконати команду у терміналі:
python -m esptool --chip esp32s3 erase-flash 

6. Відкрити термінал у каталозі Full з файлами прошивки та виконати одну із двох команд, в залежності від того, яка прошивка повинна бути основною:
python -m esptool --chip esp32s3 --baud 460800 --before default-reset --after hard-reset write-flash -z 0x0 Meow.bin
python -m esptool --chip esp32s3 --baud 460800 --before default-reset --after hard-reset write-flash -z 0x0 Keira.bin

7. Якщо основною прошивкою є Keira - скопіювати із каталогу Rollback файл Meow.bin на карту пам'яті, та використовувати в разі необхідності.
   Якщо основною прошивкою є Meow - скопіювати із каталогу Rollback файл firmware.bin в корінь карти пам'яті, та в разі необхідності запуску прошивки Keira, 
   перейти в розділ "Прошивка", натиснути та утримувати кнопку Start не менше 1 секунди. Дочекатися завершення прошивки та перезавантаження МК.
