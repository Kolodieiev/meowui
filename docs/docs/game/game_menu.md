# Створення ігрового меню
Ігрове меню це клас, який можна назвати віджетом для гри. Фактично це і є віджет, що містить в своїй основі *EmptyLayout* та *FixedMenu*. Його можна використовувати у випадках, коли в ігровій сцені потрібно відображати ігрове меню. Наприклад, якщо потрібно змінити налаштування гри без виходу з рівня, зберегти гру або завершити гру, тощо.  

Ігрове меню дуже нагадує по своєму принципу функціонування ігровий інтерфейс. Проте між ними є одна ключова відмінність. Ігровий інтерфейс може бути відрисований тільки поки гра активна, а ігрове меню тільки поки гру поставлено на паузу. Відкрийте клас *IBaseScene* та самостійно ознайомтесь, як можна налаштувати взаємодію з ігровим меню в методі *update*.
